# C++ Library Management System

## Description
This is a simple console-based Library Management System developed in C++. The project demonstrates basic Object-Oriented Programming (OOP) concepts such as classes, encapsulation, constructors, and the use of STL vectors.

## Features
- Add books to the library
- Display available books
- Issue books
- Return books

## Concepts Used
- Classes and Objects
- Encapsulation
- Constructors
- STL Vector
